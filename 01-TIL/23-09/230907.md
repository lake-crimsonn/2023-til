### 불러온 이미지에 얼굴 두개 합성하기

- `./extension/sd-webui-roop` 복사해서 하나 더 만들기 -> 2개의 스왑 얼굴 이미지를 올릴 수 있음.
- img2img -> inpaint -> 이미지 불러오기 -> generate해서 메모리에 올리기 -> 얼굴 두 개 올리기 -> enable 선택 -> generate
